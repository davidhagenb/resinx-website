# ResinX Website

## Project

Premium epoxy flooring website for ResinX, based in Hermosillo, Sonora.

## Design spec

Full spec in `docs/2026-04-12-resinx-website-design.md`

## Rules

1. No emojis anywhere in the site
2. No pricing displayed — internal info only
3. Contact placeholders use `{{VARIABLE}}` format until real values provided
4. Spanish-first, i18n-ready structure
5. All images AI-generated with fal.ai — must look 100% photorealistic
6. Mobile-first responsive
7. Vanilla HTML/CSS/JS — no frameworks

## Brand colors

- Black: #0a0a0a
- Blue: #3b82f6
- Red: #ef4444
- White: #f8fafc
- Slate: #1e293b
